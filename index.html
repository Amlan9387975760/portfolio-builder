How to get a GitHub Token (must be Classic)

  1. Go to github.com → your profile photo → Settings
  2. Scroll to bottom → Developer settings
  3. Click Personal access tokens → Tokens (classic)
  4. Click "Generate new token (classic)"
  5. Set expiration to 90 days
  6. Check these scopes: repo ✓   workflow ✓
  7. Click Generate token → Copy it immediately

⚠️ Use classic token only — fine-grained tokens will not work here.
